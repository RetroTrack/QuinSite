This mod needs to be installed client and serverside to work!

If you install it just client side, it won't work, same for just server side!

Download

How to change players?

Disclaimer: All changes made are client side only, if you want the changes to be serversided, create a serversided texture pack!
Open a text editor, like vscode, notepad++ or just notepad, and open the en_us.json file in the lang folder of the texture pack,
here you can set the players in the variables, for example  "players.hardcored.player1":"Dream"
